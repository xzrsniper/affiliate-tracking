STEP 1

Install the Tracking Pixel

Paste a small JavaScript snippet into your store's <head> tag or add it via Google Tag Manager.

  • ✓
    Copy your unique pixel code from the box below
  • 2
    Open your store admin → Themes → header.liquid (Shopify) or header.php (WooCommerce)
  • 3
    Paste the code before the closing </head> tag
  • 4
    Save and publish — the pixel fires automatically on every page
// Paste into <head> <script> (function(w,d,s,l){ w[l]=w[l]||[]; w.lhk_id = "YOUR_SITE_ID"; var f=d.createElement(s); f.src='https://cdn.lehko.track/px.js'; d.head.appendChild(f); })(window,document,'script','lehko'); </script>

STEP 2

Create Your First Tracking Link

Generate a short link that captures UTM parameters, source, medium and campaign automatically.

  • 1
    Go to Dashboard → click + New Link
  • 2
    Enter your destination URL (product page, homepage, etc.)
  • 3
    Set Source and optionally a custom alias (e.g. fb-promo)
  • 4
    Copy the short link and share it in your ad, post, or bio
# Your link looks like this: https://lhk.to/fb-promo # It redirects to: https://myshop.com/sale?utm_source=facebook &utm_medium=cpc &utm_campaign=summer25 &click_id={click_id}

STEP 3

Track Conversions via API

Send purchase and lead events from your server to attribute revenue back to each affiliate link.

  • 1
    Grab the click_id from the URL query param on arrival
  • 2
    Store it in the user session or order metadata
  • 3
    On purchase / form submit → send a POST to the Conversions API
  • 4
    Revenue, AOV, and ROAS appear in your Dashboard instantly
POST https://api.lehko.track/v1/conversion Authorization: Bearer YOUR_API_KEY { "click_id": "abc123xyz", "event": "purchase", "revenue": 149.00, "order_id": "ORD-7821" }

Frequently Asked Questions
Does the pixel slow down my store? ›
No. The pixel is loaded asynchronously and is under 2KB. It has no impact on Lighthouse or Core Web Vitals scores.
Can I track multiple websites under one account? ›
Yes. You can add multiple websites in Settings and each gets a unique Site ID. Links and analytics are separated per website.
How does the click_id work with Google Ads GCLID? ›
TrackFlow auto-captures GCLID, FBCLID, and TTCLID alongside the internal click_id so you can match conversions in both Google Ads and your own reports.
Is there a Shopify / WooCommerce plugin? ›
A native Shopify app is coming in Q3 2025. Until then, the GTM integration (one App Block or GTM snippet) works seamlessly.
