Example: Affiliate Tracker Integration

1. Basic Integration

Add this to your <head> section:

<script src="http://your-backend-url/tracker.js"></script>

2. With Configuration (Recommended)

<script>
    window.TRACKER_CONFIG = {
        BASE_URL: 'http://your-backend-url/api/track',
        DEBUG: false
    };
</script>
<script src="http://your-backend-url/tracker.js"></script>

3. Testing

To test tracking, visit your site with a referral code:

Example:
https://yourshop.com?ref=YOUR_TRACKING_CODE

4. Conversion Tracking

The tracker automatically detects conversion pages if the URL contains:

  • order
  • thank-you or thankyou
  • success
  • confirmation
  • complete
  • purchase

Example conversion URLs:

  • https://yourshop.com/order/12345
  • https://yourshop.com/thank-you
  • https://yourshop.com/purchase/success

5. Manual Tracking (Advanced)

You can manually trigger tracking if needed:

// Track a view manually
window.AffiliateTracker.trackView();

// Track a conversion manually
window.AffiliateTracker.trackConversion();

// Get current visitor ID
const visitorId = window.AffiliateTracker.getVisitorId();

// Get stored referral code
const refCode = window.AffiliateTracker.getRefCode();
