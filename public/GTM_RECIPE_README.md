# GTM Recipe: Lehko Track Universal

Файл **gtm-recipe-lehko-universal.json** — готовий рецепт для імпорту в Google Tag Manager.

## Як імпортувати

1. Відкрийте Google Tag Manager → **Admin** → **Import Container**.
2. Виберіть файл `gtm-recipe-lehko-universal.json`.
3. Оберіть **Merge** (об'єднати) або **Overwrite** (перезаписати) за потреби.
4. Після імпорту відкрийте тег **Lehko Track Universal Pixel** і замініть `YOUR_SITE_ID` на **ID вашого сайту** з розділу «Мої сайти» в кабінеті Lehko Track (число, наприклад `1` або `2`).

## Що входить у рецепт

- **Один тег:** Custom HTML — завантажує `pixel.js?id=SITE_ID` на всіх сторінках.
- **Тригер:** All Pages (перегляд сторінки).

Після публікації контейнера Universal трекер буде:

- Відправляти кліки по посиланнях на сервер.
- Перевіряти URL сторінки на збіг з налаштованими «URL успіху» (конверсія).
- Відстежувати кліки по кнопках типу «Купити», «Замовити», «Оплатити».
- Підхоплювати події purchase/conversion з dataLayer.

## Налаштування конверсій

У кабінеті Lehko Track: **Налаштування** → **Мої сайти** → кнопка **Налаштування** (іконка слайдерів) біля потрібного сайту. Там можна вказати:

- **URL сторінки успіху** (наприклад `/thanks`, `/success`) — по одному на рядок.
- **CSS-селектор ціни** (наприклад `.product-price`) — для скрапінгу суми зі сторінки.
- **Статична ціна** — фіксована сума за конверсію, якщо ціни немає на сторінці.

Ці налаштування підхоплює Universal трекер автоматично за параметром `id` у скрипті (`pixel.js?id=...`).
